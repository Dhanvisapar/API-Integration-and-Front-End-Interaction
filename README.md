# API-Integration-and-Front-End-Interaction

## Objective
To demonstrate server-client communication using a RESTful API and front-end interaction.

## Features
- RESTful API with CRUD operations
- Fetch API for front-end communication
- Dynamic UI updates
- Client-server interaction

## API Endpoints
- GET /api/items
- POST /api/items
- PUT /api/items/:id
- DELETE /api/items/:id

## How to Run
1. Open folder in VS Code
2. Open terminal
3. Run:
   npm install
   npm start
4. Open browser:
   http://localhost:3000

## Learning Outcomes
- REST API design
- Fetch API usage
- Client-server communication
